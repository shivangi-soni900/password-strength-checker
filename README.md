# 🔐 Password Strength Checker

A simple cybersecurity project that evaluates the strength of a password using Python.

## Features

- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters
- Gives security suggestions

## Technologies Used

- Python
- Regular Expressions

## Installation

Clone the repository

git clone https://github.com/yourusername/password-strength-checker.git

Navigate into the folder

cd password-strength-checker

Run the program

python password_checker.py

## Example

Input:
Password123

Output:
Password Strength: Strong

Suggestions:
Add at least one special character

## Why This Project?

Weak passwords are one of the biggest cybersecurity risks. This project demonstrates how password policies can improve account security.
